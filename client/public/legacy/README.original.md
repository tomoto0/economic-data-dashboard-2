# 経済データダッシュボード - モダン版

## 概要
世界銀行オープンデータを活用した、モダンでインタラクティブな経済データダッシュボードです。各経済指標を国別に比較し、時系列での推移を動的に表示できます。

## デプロイ先
このダッシュボードはGitHub Pagesにデプロイされており、以下のURLからアクセスできます。
[https://tomoto0.github.io/economic-data-dashboard/](https://tomoto0.github.io/economic-data-dashboard/)

## 特徴
- 📊 インタラクティブなチャート表示：GDP、人口、インフレ率、失業率、外国直接投資、外貨準備高を国別・指標別に動的に切り替え可能。
- 🌍 国別サマリーカード：選択した国の主要経済指標を一覧表示。
- 🌙 ダークモード対応
- 📱 レスポンシブデザイン
- 🔄 自動データ更新：GitHub Actionsにより毎日午後4時（日本時間）に自動更新されます。
- 🎨 モダンなUI/UX

## 表示データ
- GDP (current US$)
- 人口
- インフレ率
- 失業率
- 外国直接投資
- 外貨準備高

## 対象国
USA, CHN, JPN, DEU, GBR, FRA, IND, ITA, BRA, CAN

## 技術スタック
- HTML5 + CSS3 + JavaScript
- Chart.js (データ可視化)
- Tailwind CSS (スタイリング)
- GitHub Actions (自動更新)
- GitHub Pages (ホスティング)
